Dokumentasi Bahasa Pemrograman

Berikut adalah penjelasan dan contoh kode dari berbagai bahasa pemrograman.

1. HTML (HyperText Markup Language)

Penjelasan: HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

Struktur Dasar HTML:
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Halaman HTML</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

2. CSS (Cascading Style Sheets)

Penjelasan: CSS digunakan untuk mengatur tampilan dan desain halaman HTML.

CSS Dasar:
body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}

3. JavaScript

Penjelasan: JavaScript digunakan untuk membuat halaman web lebih interaktif.

Menampilkan Pesan di Konsol:
console.log("Hello, World!");

4. Python

Penjelasan: Python adalah bahasa pemrograman serbaguna yang digunakan dalam berbagai bidang.

Hello, World!:
print("Hello, World!")

5. C

Penjelasan: C adalah bahasa pemrograman tingkat rendah yang digunakan untuk sistem operasi dan embedded systems.

Hello, World! dalam C:
#include <stdio.h>
int main() {
    printf("Hello, World!\n");
    return 0;
}

6. C++

Penjelasan: C++ adalah versi pengembangan dari C yang mendukung pemrograman berbasis objek.

Hello, World! dalam C++:
#include <iostream>
using namespace std;
int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

7. Java

Penjelasan: Java adalah bahasa pemrograman berbasis objek yang berjalan di berbagai platform.

Hello, World! dalam Java:
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

8. PHP

Penjelasan: PHP adalah bahasa server-side yang digunakan untuk membuat situs web dinamis.

Hello, World! dalam PHP:
<?php
echo "Hello, World!";
?>

9. Ruby

Penjelasan: Ruby adalah bahasa pemrograman yang terkenal dengan framework Ruby on Rails.

Hello, World! dalam Ruby:
puts "Hello, World!"

10. Go

Penjelasan: Go adalah bahasa pemrograman dari Google yang dikenal dengan kecepatan dan efisiensinya.

Hello, World! dalam Go:
package main
import "fmt"
func main() {
    fmt.Println("Hello, World!")
}

11. Swift

Penjelasan: Swift adalah bahasa pemrograman dari Apple untuk membuat aplikasi iOS dan macOS.

Hello, World! dalam Swift:
print("Hello, World!")

12. Rust

Penjelasan: Rust adalah bahasa pemrograman yang aman dan cepat untuk sistem tingkat rendah.

Hello, World! dalam Rust:
fn main() {
    println!("Hello, World!");
}

13. Perl

Penjelasan: Perl sering digunakan untuk pemrosesan teks dan skrip administrasi.

Hello, World! dalam Perl:
print "Hello, World!\n";
