14. Dart

Penjelasan: Dart adalah bahasa pemrograman yang dikembangkan oleh Google dan digunakan terutama untuk pengembangan aplikasi Flutter.

Hello, World! dalam Dart:
void main() {
    print("Hello, World!");
}
Looping dan Kondisi dalam Dart:
void main() {
    for (int i = 0; i < 5; i++) {
        if (i % 2 == 0) {
            print("$i adalah bilangan genap");
        } else {
            print("$i adalah bilangan ganjil");
        }
    }
}
Fungsi dalam Dart:
String sapa(String nama) {
    return "Halo, $nama!";
}

void main() {
    print(sapa("Budi"));
}

15. Haskell

Penjelasan: Haskell adalah bahasa pemrograman fungsional yang terkenal dengan sistem tipe yang kuat dan imutabilitas.

Hello, World! dalam Haskell:
main :: IO ()
main = putStrLn "Hello, World!"
Looping dan Kondisi dalam Haskell:
evenOdd :: Int -> String
evenOdd n = if even n then show n ++ " adalah genap" else show n ++ " adalah ganjil"

main :: IO ()
main = mapM_ (putStrLn . evenOdd) [0..4]
Fungsi dalam Haskell:
sapa :: String -> String
sapa nama = "Halo, " ++ nama ++ "!"

main :: IO ()
main = putStrLn (sapa "Budi")

16. SQL

Penjelasan: SQL (Structured Query Language) adalah bahasa yang digunakan untuk mengelola dan memanipulasi basis data relasional.

Membuat Tabel dalam SQL:
CREATE TABLE pengguna (
    id INT PRIMARY KEY,
    nama VARCHAR(100),
    email VARCHAR(100)
);
Menambahkan Data dalam SQL:
INSERT INTO pengguna (id, nama, email) VALUES (1, 'Budi', 'budi@example.com');
Mengambil Data dalam SQL:
SELECT * FROM pengguna;

17. Bash

Penjelasan: Bash adalah bahasa skrip yang digunakan dalam sistem operasi berbasis Unix dan Linux untuk mengotomatisasi tugas.

Hello, World! dalam Bash:
#!/bin/bash
echo "Hello, World!"
Looping dan Kondisi dalam Bash:
for i in {0..4}; do
    if ((i % 2 == 0)); then
        echo "$i adalah bilangan genap"
    else
        echo "$i adalah bilangan ganjil"
    fi
done
Fungsi dalam Bash:
sapa() {
    echo "Halo, $1!"
}

sapa "Budi"
